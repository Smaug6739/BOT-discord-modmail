# Discord mod-mail-bot

A simple discord mod-mail bot.

## How to install
1. Clone the repo
3. Clone `template.config.js` into the main directory and rename it to `config.js`
2. Install the dependencies: `npm install`
3. Start the bot: `node main.js`
